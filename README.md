Introduction to Beginner R Programing

Here you will learn everything from basic R commands! Below you will find some steps to help get you comfortable with R Studion

## Basics
Within R, you are able to preform basic mathematical functions.
```
# Calculate 3 + 4
3 + 4

# Calculate 6 + 12
6 + 12

# An addition
5 + 5 

# A subtraction
5 - 5 

# A multiplication
3 * 5

 # A division
(5 + 5) / 2 

# Exponentiation
2^5

# Modulo
28 %% 6
```
